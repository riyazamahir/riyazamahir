About Web Learning

Web learning is an essential skill in the digital world, helping you understand the fundamentals of designing and developing websites.

HTML (Hypertext Markup Language)

HTML is the standard language for creating web pages. It provides the structure of the web content.


<div>
    <h1>Welcome</h1>
    <p>This is a sample HTML content.</p>
</div>

CSS (Cascading Style Sheets)

CSS is used for styling the visual presentation of web pages. It controls the layout, colors, and fonts.


body {
    background-color: #f4f4f4;
}
button {
    background-color: #333;
    color: white;
}

JavaScript

JavaScript is a programming language that enables dynamic interactivity on web pages.


document.querySelector('button').addEventListener('click', function() {
    alert('Button clicked!');
});

Responsive Design

Responsive design ensures that your website looks great on all devices by adjusting layouts for different screen sizes.


@media screen and (max-width: 768px) {
    body {
        background-color: lightblue;
    }
}
